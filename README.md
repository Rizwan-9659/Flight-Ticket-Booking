# Flight-Ticket-Booking
Html css javascript php 

1️⃣ User Registration & Login

Passengers can create an account and log in securely.

Passwords are stored securely using hashing (optional but recommended).

2️⃣ Search Flights

Users can search flights based on:

Source

Destination

Travel date

Number of passengers

3️⃣ View Flight Details

Displays available flights with:

Flight number

Airline name

Timing

Price

Remaining seats

4️⃣ Book Tickets

Users can select a flight and complete the booking.

System checks seat availability before booking.

Booking details are stored in the database.

5️⃣ View / Download Ticket

Users can see their booking history.

Ticket includes flight details + passenger information.
